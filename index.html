OPA-Java SDK Documentation Site

This site contains the documentation for opa-java, a Java SDK for the Open Policy Agent REST API. "Low level" wrappers around the REST API methods are generated using Speakeasy. A higher level human-written API is also provided, which is intended to simplify the most common tasks for OPA API consumers. The Speakeasy-generated API can be found in the com.styra.opa.sdk package, while the higher level API is located in com.styra.opa.

Helpful links:

  • StyraInc/opa-java GitHub repository
  • opa-java JavaDoc
  • Speakeasy generated "models" documentation
  • Speakeasy generated "SDKs" documentation
  • OpenAPI Spec for the OPA REST API
