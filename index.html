OPA-Java SDK Developer Documentation Site

This site is meant to assist developers of the opa-java project itself. Looking for docs on how to use it? Consider the JavaDoc or SDK docs.

This site contains the developer documentation for opa-java, a Java SDK for the Open Policy Agent REST API. "Low level" wrappers around the REST API methods are generated using Speakeasy. A higher level human-written API is also provided, which is intended to simplify the most common tasks for OPA API consumers. The Speakeasy-generated API can be found in the com.styra.opa.sdk package, while the higher level API is located in com.styra.opa.
